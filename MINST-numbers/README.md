### Comparison of other approaches

We have previously explored **Hyperdimensional Computing (HDC)**, **Predictive Coding (PC)** and **Reservoir Computing (RC)** for these types of tasks. Compared to those:


| Feature | Reservoir Computing | HDC (Vector Symbolic) | Predictive Coding |
| --- | --- | --- | --- |
| **Logic** | Dynamic systems/Chaos | Algebraic high-dim vectors | Error minimization/Inference |
| **Training** | Linear Regression (Fast) | Single-pass (Very Fast) | Backprop or local rules (Slower) |
| **Best For** | Temporal/Sequential data | Robust classification/Logic | Biological modeling |
