### Comparison of other approaches

We have previously explored **Hyperdimensional Computing (HDC)**, **Predictive Coding (PC)**, **Reservoir Computing (RC)** and **Joint-Embedding Predictive Architecture (JEPA)** for these types of tasks. Compared to those:



| Feature | Reservoir Computing | HDC (Vector Symbolic) | Predictive Coding | JEPA (Joint-Embedding) |
| --- | --- | --- | --- | --- |
| **Logic** | Dynamic systems / Chaos | Algebraic high-dim vectors | Error minimization / Inference | Latent structure prediction / Representation alignment |
| **Training** | Linear Regression (Fast) | Single-pass (Very Fast) | Backprop or local rules (Slower) | Contrastive-free Backprop / EMA target update (Moderate) |
| **Best For** | Temporal / Sequential data | Robust classification / Logic | Biological modeling | Sample-efficient feature learning / Common-sense world models |
